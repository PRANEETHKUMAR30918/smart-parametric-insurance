# Frontend - SentinelFlow

The frontend layer of SentinelFlow is responsible for providing an intuitive interface for gig workers to monitor risks, view claim status, and interact with the system.

## Responsibilities

* Display real-time risk alerts based on environmental conditions
* Show claim status and payout information
* Provide a simple dashboard for user interaction

## Key Components

* **App.js** – Entry point of the application
* **Dashboard.js** – Displays overall system insights
* **RiskView.js** – Shows current risk level and alerts

## Tech Stack

* React.js

## Future Enhancements

* Real-time notifications
* Interactive dashboards
* Integration with backend APIs for live data
